## Pré-requisitos
`Python` | `pip`
## Criar ambiente de desenvolvimento
Linux
```
$ python3 -m venv venv
```
Windows
```
python -m venv venv
```

## Ativar ambiente de desenvolvimento
Linux
```
$ source venv/bin/activate
```
Windows
```
venv/Scripts/activate.bat
```

## Instalar dependências
```
$ pip install -r requirements.txt
```

## Iniciar projeto
Criar banco de dados
```
flask db upgrade
```
Iniciar a aplicação
```
flask run
```
